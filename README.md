# My Portfolio

A professional portfolio website showcasing my skills, projects, and achievements as a Full Stack Developer.

## 🌐 Live Demo
[Visit My Portfolio](https://ishank107.github.io/My-Portfolio/)

## ✨ Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations** - Scroll-triggered reveal animations for engaging UX
- **Projects Showcase** - Featured projects with descriptions and technologies
- **Skills Section** - Organized by categories (Languages, Frameworks, Databases, DevOps)
- **Certifications** - Display of achievements and professional certifications
- **Contact Section** - Easy ways to connect and collaborate

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Animations**: CSS Transitions + Intersection Observer API
- **Deployment**: GitHub Pages
- **Version Control**: Git

## 📂 Project Structure
├── index.html # Main portfolio page
├── style.css # All styling and animations
├── main.js # JavaScript for interactions
├── package.json # Project configuration
├── my.jpg # Profile image
└── certifications/ # Certificate screenshots

Run Locally
Opens at http://localhost:8000

📝 Sections
Hero - Introduction and call-to-action
About - Background and professional summary
Skills - Technical expertise organized by category
Projects - Featured work and contributions
Achievements - Certifications and recognitions
Education - Academic background
Contact - Email and social links
🎨 Customization
Edit index.html to add/modify content, style.css for styling, and main.js for interactions.
